# Coding Merah Putih – CS50 Problem Sets

Welcome to the Coding Merah Putih Problem Sets repository.

This repository contains programming exercises adapted from the CS50 curriculum for students.

---

## How to Download All Problem Sets

Run this in the terminal:

wget https://github.com/Aristo77/Problem-sets/archive/refs/heads/main.zip -O problem-sets.zip

unzip problem-sets.zip

cd Problem-sets-main

---

## Available Problem Sets

- mario-less
- cash
- caesar
- sort
- filter-less
- birthdays
- dna
- inheritance
- finance

---

## How to Run a Problem Set

Example:

cd mario-less

make mario

./mario

---

## Notes for Students

- Each folder contains one programming problem
- Read the instructions carefully
- Compile using `make`
- Run the program in the terminal
